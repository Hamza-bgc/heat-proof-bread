# Heat-Proof Bread

A mobile-first interactive story on why bread hardens too fast in summer heat, and the six technical adjustments that fix it — hydration, fermentation, autolyse, improver, baking, and cooling.

Built for **Saf Djazair × Lesaffre**, sourced from verified Lesaffre technical content.

**Live:** https://hamza-bgc.github.io/heat-proof-bread/

## Features
- Auto-advancing scene carousel (8s per scene) with manual next/back controls
- Swipe navigation on touch devices, arrow-key navigation on desktop
- Responsive scaling — fills the screen on any device
- Open Graph / Twitter Card previews for clean link sharing on LinkedIn
- Respects reduced-motion accessibility preference

## Structure
```
index.html          — the full story (self-contained HTML/CSS/JS)
assets/og-image.jpg  — social share preview image
assets/favicon-*.png — favicons
```

No build step. No dependencies. Just static files served via GitHub Pages.
